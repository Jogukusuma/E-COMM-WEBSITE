# **App Name**: SriComputers

## Core Features:

- Product Catalog: Display a catalog of computers, laptops, and hardware components with images, descriptions, and pricing.
- Shopping Cart: Allow users to add computers, laptops, and hardware components to a shopping cart and view/modify its contents.
- Checkout Process: Enable users to complete purchases of computers, laptops, and hardware components with payment processing integration.
- AI Product Recommendations: Suggest related or complementary computers, laptops, and hardware components to users based on browsing history and cart contents using a tool.
- User Accounts: Allow users to create accounts for order tracking and saved information.
- Order History: Enable users to view their past orders of computers, laptops, and hardware components.
- Search Functionality: Enable users to search for computers, laptops, and hardware components by keyword or category.
- Theme Toggle: Allow users to toggle between light, dark, and system default themes.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of sophistication and trust.
- Background color: Light beige (#F5F5DC).
- Accent color: Coral (#FF7F50) to draw attention to calls to action, and add a playful vibrancy.
- Body and headline font: 'Inter' for a modern, clean look.
- Use minimalist icons to represent product categories and cart functions.
- Grid-based layout to showcase products clearly and attractively.
- Subtle animations on product hover and when adding items to the cart.
